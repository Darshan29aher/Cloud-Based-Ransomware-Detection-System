📋 About Project

Cloud-Based Ransomware Detection & Monitoring System is a security application that continuously monitors protected directories for suspicious ransomware behaviour. The system detects unauthorized file modifications, logs security events, generates incident reports, sends email notifications, and integrates with cloud services for secure backup and centralized monitoring.

✔ Key Features
  • Real-time File Monitoring
  • Behavior-based Ransomware Detection
  • SQLite Event Logging
  • Automatic PDF Report Generation
  • Email Alert Notifications
  • Cloud Storage Backup
  • Cloud Logging Integration
  • Interactive Dashboard
  • Threat Analytics using Chart.js
